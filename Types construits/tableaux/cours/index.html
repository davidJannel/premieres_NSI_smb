    CH7 : Tableaux⚓︎

    Programme officiel

    1. Tableaux en Python⚓︎

    1.1 Qu'est ce qu'un tableau ?⚓︎

    À connaître

    Les tableaux, ou encore appelés en Python les listes, sont des séquences (ensemble ordonné d'éléments indicés). Une variable de type list sera délimité par des crochets, et ses éléments séparés par des virgules.

    >>> t = [7, 8, 12]
>>> type(t)
<class 'list'>

    Remarque :

    Une liste vide se déclare avec [] ou avec l'appel de fonction list().

    1.2 Accés aux éléments d'un tableau⚓︎

    Savoir faire

    On accède aux éléments d'un tableau en mettant entre crochets l'indice de l'élément. Le premier élément a pour indice 0.

    >>> notes = [6, 12, 15, 11]
>>> notes[0]
6
>>> notes[3]
11
>>> len(notes) # len renvoie la longueur du tableau
4 
>>> notes[-1] # accés par indice négatif comme pour les tuples
11

    ```

    2. Parcourir les éléments d'un tableau⚓︎

    2.1 Parcours par éléments⚓︎

    Savoir faire

    On utilise une boucle for en itérant sur les éléments en utilisant le mot clé in.

    >>> noms = ['toto', 'tata', 'titi']
>>> for elt in noms:
	print(elt)
toto
tata
titi

    2.2 Parcours par indice⚓︎

    Savoir faire

    On utilise toujours une boucle for, mais on itére sur les indices en créant un itérable à l'aide de la fonction range. Pour un tableau de longueur n, range(len(tableau)) renvoie un itérable allant de 0 à n-1 par pas de 1.

    >>> noms = ['toto', 'tata', 'titi']
>>> for i in range(len(noms)):
	print(noms[i])
toto
tata
titi

    3. Modification d'un tableau⚓︎

    3.1 Modification des éléments d'un tableau⚓︎

    À savoir

    Contrairement aux tuples (immutables), on peut modifier les éléments d'un tableau. Un objet de type list est dit mutable.

    >>> notes = [6, 12, 15, 11]
>>> notes[0] = 17
>>> print(notes)
[17, 12, 15, 11]

    3.2 Ajout d'un élément à la fin d'un tableau : append⚓︎

    À savoir

    La méthode append permet d'ajouter un élément à la fin d'un tableau existant.

    >>> notes_nsi = [16, 18]
>>> notes_nsi.append(19)
>>> print(notes_nsi)
[16, 18, 19]

    Remarques :

    • Il est possible d'insérer un élément à la position i en utilisant la méthode insert :

    >>> notes_nsi = [16, 18, 19]
>>> notes_nsi.insert(1, 17) # on insère 17 à la position 1
>>> print(notes_nsi)
[16, 17, 18, 19]
    - On peut également concaténer des tableaux (mais c'est pas vraiment une insertion, on a créé un nouveau tableau) :

    >>> t1 = [7, 8]
>>> t2 = [9]
>>> t3 = t1 + t2
>>> print(t3)
[7, 8, 9]

    3.3 Suppression d'un élément d'une liste : remove et del⚓︎

    Savoir faire

    Deux méthodes possibles :

    • remove enlève la première occurrence de l'élément désigné. S'il y en a d'autres après, elles resteront dans la liste :
    >>> lst = [3, 1, 4, 5, 1, 9, 4]
>>> lst.remove(4)
>>> lst
[3, 1, 5, 1, 9, 4]
    • del permet de supprimer un élément en donnant son indice :
    >>> maliste = [8, 4, 2, 5, 7]
>>> del maliste[3]
>>> maliste
[8, 4, 2, 7]

    3.4 Extraire l'élément d'un tableau : pop⚓︎

    Savoir faire

    La méthode pop permet de récupérer et de supprimer un élément d'un tableau. Sans argument, cette méthode renvoie et supprime le dernier élément du tableau.

    >>> maliste = [8, 4, 2, 5, 7]
>>> elt = maliste.pop()
>>> print(elt)
7
>>> print(maliste)
[8, 4, 2, 5]

    Avec en argument un indice :

    >>> premier_elt = maliste.pop(0)
>>> print(premier_elt)
8
>>> print(maliste)
[4, 2, 5]

    4. Construction d'un tableau en compréhension⚓︎

    Savoir faire

    La construction du tableau se fait avec la syntaxe suivante par exemple :

    >>> tab = [2**i for i in range(4)] # construction d'un tableau contenant les 4 premiers nombres pairs.
>>> print(tab)
[0, 2, 4, 6]
    On peut également ajouter des filtres comme dans l'exemple suivant :

    >>> temp = [17, 22, 15, 28, 16, 13, 21, 23] # liste de temperatures
>>> chaud = [t for t in temp if t > 20] # liste des temperatures superieures a 20 degre
>>> print(chaud)
[22, 28, 21, 23]

    5. Tableau à plusieurs dimensions⚓︎

    Un objet de type list peut contenir tout type d'éléments. Et donc on peut imaginer un objet de type list contenant des objets de type list. On a des tableaux de tableaux.

    Savoir faire

    La liste tab ci-dessous est composée de 3 listes qui elles-mêmes contiennent trois nombres : 

    tab =  [[3, 5, 2],
        [7, 1, 4], 
        [8, 6, 9]]

    • tab[0][0] = 3
    • tab[0][1] = 5
    • tab[2][1] = 6
    • tab[1] = [7, 1, 4]

    6. Programmation : la copie de liste⚓︎

    une copie un peu trop parfaite

    Observez le code ci-dessous, réalisé sans trucage. 

    >>> listA = [1, 2, 3]
>>> listB = listA
>>> listA.append(7)
>>> listB
[1, 2, 3, 7]
>>> listB.append(8)
>>> listA
[1, 2, 3, 7, 8]

    Tout se passe comme si les listes listA etlistB étaient devenus des clones «synchronisés» depuis l'affectation listB = listA.

    Analyse grâce à PythonTutor

    L'illustration de PythonTutor nous donne la clé de l'énigme :

    listA etlistB sont en fait un seul et même objet. Comment en avoir le cœur net ? En observant leur adresse-mémoire, disponible grâce à la fonction id : 

    >>> id(listA)
140485841327616
>>> id(listB)
140485841327616

    Ceci met en évidence que la métaphore du tiroir dont on se sert pour expliquer ce qu'est une variable est malheureusement inexacte. Une variable est une référence vers une adresse-mémoire. Si deux variables font référence à la même adresse-mémoire, alors elles sont totalement identiques: toute modification de l'une entraîne une modification de l'autre.

    Mais alors, comment copier le contenu d'une liste vers une autre sans créer un clone ?⚓︎

    Exemple

    >>> listA = [3, 4, 5]
>>> listB = list(listA)
    D'autres possibilités existent, comme listA.copy(), ou encore listA[::]...
